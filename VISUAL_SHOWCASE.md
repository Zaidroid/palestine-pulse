## 🎬 Interaction Flow

### Expandable Card Flow

```
1. Initial State
   ┌─────────────┐
   │ Title       │
   │ 45,000      │
   │ Quick Info  │
   └─────────────┘

2. Click Card
   ┌─────────────┐
   │ Title       │
   │ 45,000      │
   │ Quick Info  │
   │ ─────────── │
   │ More Info   │ ← Expands
   │ [Details]   │
   └─────────────┘

3. Click "View Full Details"
   ╔═══════════════════╗
   ║ Modal Overlay     ║
   ║ ┌───────────────┐ ║
   ║ │ Full Details  │ ║
   ║ │               │ ║
   ║ │ All Info      │ ║
   ║ │               │ ║
   ║ └───────────────┘ ║
   ╚═══════════════════╝

4. Click Outside or X
   → Returns to Step 2
```

### Data Source Badge Flow

```
1. Rest State
   [🗄️ Source: Name ✓ ℹ️]

2. Hover
   [🗄️ Source: Name ✓ ℹ️]
          ↓
   ┌─────────────────┐
   │ Details Panel   │
   │ • Last Updated  │
   │ • Reliability   │
   │ • Methodology   │
   │ • Link          │
   └─────────────────┘

3. Mouse Leave
   → Returns to Step 1
```
